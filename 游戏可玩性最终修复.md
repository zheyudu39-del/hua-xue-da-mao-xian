## 最终解决方案

### 使用固定安全坐标

```javascript
this.terrain = new Terrain();
this.panda = new Panda(300, 170); // Y=170，在地形baseY=200上方
```

### 为什么选择Y=170？

```
地形起点：baseY = 200
人物高度：30px
人物Y坐标：170

人物顶部：Y = 170
人物脚底：Y = 170 + 30 = 200
地形起点：Y = 200

初始间隙：200 - 200 = 0px（刚好接触）
```

### 实际效果

```
Y=170  ┌──┐ ← 人物顶部
       │👤│
Y=200  └──┘━━━━ ← 人物脚底 = 地形起点
       ╲
        ╲ 坡面向下
         ╲
```
