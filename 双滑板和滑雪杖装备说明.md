# 双滑板和滑雪杖装备系统说明

## 功能概述
为滑雪者添加了完整的专业滑雪装备：双滑雪板 + 双滑雪杖，使角色更加真实和专业。

## 装备详情

### 1. 双滑雪板 🎿🎿

#### 左滑雪板
- **位置**：左脚下方
- **尺寸**：40×6像素
- **颜色**：红色渐变（#FF6B6B → #FF4757 → #C23616）
- **特效**：
  - 阴影效果
  - 高光反射
  - 白色装饰条纹
  - 黑色边缘轮廓

#### 右滑雪板
- **位置**：右脚下方
- **尺寸**：40×6像素
- **颜色**：与左滑雪板相同
- **特效**：
  - 对称的阴影效果
  - 高光反射
  - 白色装饰条纹
  - 黑色边缘轮廓

#### 3D立体效果

### 2. 双滑雪杖 🏒🏒

#### 左滑雪杖（前手）
**组成部分**：

1. **握把**（顶部）
   - 颜色：深灰色 (#2C2C2C)
   - 尺寸：4×8像素
   - 边框：红色装饰 (#FF4757)

2. **杆身**（主体）
   - 长度：35像素
   - 材质：金属渐变质感
   - 颜色：银灰色渐变 (#A0A0A0 → #D0D0D0 → #808080)
   - 粗细：2.5像素
   - 端点：圆形

3. **尖端**（底部）
   - 形状：三角形
   - 颜色：深灰色 (#404040)
   - 长度：5像素

4. **雪圈**（底部圆盘）
   - 形状：椭圆形
   - 尺寸：5×2像素
   - 颜色：半透明红色 (rgba(255, 107, 107, 0.8))
   - 边框：深红色 (#C23616)
   - 作用：防止滑雪杖陷入雪中

#### 右滑雪杖（后手）
- 结构与左滑雪杖相同
- 金属渐变略有不同（更暗的色调）
- 位置和角度对称

### 3. 动态效果

#### 跳跃时的动作
```javascript
// 手臂摆动
const armSwing = this.jumpPose * 5;

// 滑雪杖角度调整
const poleAngle = this.jumpPose * 0.2;
```

**上升阶段**（jumpPose < 0）：
- 手臂向后摆
- 滑雪杖向后倾斜
- 呈现蓄力姿态

**最高点**（jumpPose ≈ 0）：
- 手臂水平
- 滑雪杖垂直
- 平衡姿态

**下降阶段**（jumpPose > 0）：
- 手臂向前伸
- 滑雪杖向前倾斜
- 准备着陆姿态

### 地面滑行
```
手臂：自然下垂
滑雪杖：垂直向下
滑雪板：贴合地面
```

### 跳跃上升
```
手臂：向后摆动
滑雪杖：向后倾斜 (-0.6 - poleAngle)
滑雪板：离开地面
```

### 空中最高点
```
手臂：水平展开
滑雪杖：接近垂直
滑雪板：保持水平
```

### 下降着陆
```
手臂：向前伸展
滑雪杖：向前倾斜 (-0.6 + poleAngle)
滑雪板：准备接触地面
```
