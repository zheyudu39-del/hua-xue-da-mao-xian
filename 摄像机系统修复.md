# 摄像机系统修复 📹

## 问题原因

之前人物Y=170太靠上，摄像机跟随系统会将画面向上推，导致：
- 坡面显示在屏幕底部
- 白色雪层看不见
- 画面布局错误

## 摄像机系统工作原理

```javascript
// 摄像机目标：保持人物在屏幕50%处
const targetY = CONFIG.canvas.height * 0.5; // 1080px
const pandaScreenY = this.panda.y - this.cameraOffsetY;
const offsetDiff = pandaScreenY - targetY;
this.cameraOffsetY += offsetDiff * 0.03; // 平滑跟随
```

### 计算示例

**修复前（Y=170）：**
```
人物Y: 170
目标屏幕Y: 1080 (50%)
摄像机偏移: 170 - 1080 = -910
画面向上推910px → 坡面在屏幕底部 ❌
```

**修复后（Y=900）：**
```
人物Y: 900
目标屏幕Y: 1080 (50%)
摄像机偏移: 900 - 1080 = -180
画面向上推180px → 坡面在正确位置 ✅
```

## 最终修复

```javascript
// 游戏初始化和重启
this.panda = new Panda(300, 900); // Y=900
```

## 刷新浏览器

按 **F5**，您应该看到：
- ✅ 坡面在正确位置
- ✅ 白色雪层可见
- ✅ 人物在屏幕中间
- ✅ 画面布局正确
