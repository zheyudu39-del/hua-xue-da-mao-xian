# 小屋前后门功能说明

## 功能概述
小屋现在配备了前门和后门，两扇门都是打开的状态，玩家（熊猫）可以直接穿过小屋而不会发生碰撞。

## 实现细节

### 1. 前门（左侧入口）
- **位置**：小屋正面左侧（`this.x + 20`）
- **尺寸**：宽30px，高50px
- **设计特点**：
  - 深色木质渐变效果（`#5A3A1F` → `#4A2F1A` → `#3A2515`）
  - 金色门把手（`#FFD700`）
  - 木纹纹理（4条横向木板线）
  - 深色门框轮廓

### 2. 后门（右侧出口）
- **位置**：小屋正面右侧（`this.x + this.width - 50`）
- **尺寸**：宽30px，高50px
- **设计特点**：
  - 较亮的木质渐变效果（`#6A4A2F` → `#5A3F25` → `#4A3420`）
  - 金色门把手（`#FFD700`）
  - 木纹纹理（4条横向木板线）
  - 深色门框轮廓

### 3. 窗户（中间位置）
- **位置**：居中于两扇门之间
- **尺寸**：宽30px，高25px
- **设计特点**：
  - 温暖的金黄色玻璃反光效果
  - 十字窗框分隔
  - 3D立体效果

## 碰撞检测逻辑

在 `game.js` 第220-223行实现了特殊的碰撞检测逻辑：

```javascript
// 小屋：完全可以穿过，不产生碰撞
if (obs.type === 'house') {
    return true; // 保留小屋，不检测碰撞
}
```

这确保了：
- 小屋不会被移除
- 玩家可以自由穿过小屋
- 不会触发任何碰撞事件

## 视觉效果

### 前门
- 作为入口，使用较暗的颜色表示深度
- 门把手位于右侧（`this.x + 43`）
- 木纹横向排列，增强真实感

### 后门
- 作为出口，使用较亮的颜色
- 门把手位于左侧（`this.x + this.width - 27`）
- 与前门形成视觉对比

### 整体布局
```
[前门] ---- [窗户] ---- [后门]
  左侧        中间        右侧
  入口                   出口
```

## 游戏玩法
1. 玩家从前门进入小屋
2. 穿过小屋内部（无碰撞）
3. 从后门离开小屋
4. 整个过程流畅自然，不影响游戏节奏
