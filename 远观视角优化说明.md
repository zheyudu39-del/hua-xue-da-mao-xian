# 🔭 远观视角优化说明

## 核心改进

### 从近景视角改为远观视角
- 缩小所有物体到原尺寸的60%
- 让玩家能看到更多场景内容
- 获得更宏观的游戏视野

---

## 技术实现

### 代码修改：

```javascript
// 熊猫
class Panda {
    constructor(x, y) {
        // 近景：50×50
        // 远观：30×30
        this.width = 30;
        this.height = 30;
    }
}

// 障碍物
class Obstacle {
    constructor(x, y, type, terrain) {
        // 近景：石头90×75，小屋220×180
        // 远观：石头54×45，小屋132×108
        this.width = type === 'rock' ? 54 : 132;
        this.height = type === 'rock' ? 45 : 108;
    }
}

// 动物
class Animal {
    constructor(x, y, type, terrain) {
        // 近景：企鹅60×75，北极熊90×75
        // 远观：企鹅36×45，北极熊54×45
        this.width = type === 'penguin' ? 36 : 54;
        this.height = type === 'penguin' ? 45 : 45;
    }
}
```
