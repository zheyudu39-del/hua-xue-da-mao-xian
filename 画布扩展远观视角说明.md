# 画布扩展远观视角优化说明

## 功能概述
将游戏画布从1600×900扩展到1920×1080（Full HD），并调整摄像机位置，让玩家能够看到更多的游戏场景。

## 核心改动

### 1. 画布尺寸扩展
```javascript
// 修改前（近景）
canvas: {
    width: 1600,
    height: 900
}

// 修改后（远观）
canvas: {
    width: 1920,
    height: 1080
}
```

**变化**：
- 宽度：1600px → 1920px（+20%）
- 高度：900px → 1080px（+20%）
- 总面积：1,440,000px² → 2,073,600px²（+44%）

### 2. 摄像机位置调整
```javascript
// 修改前（近景）
const targetY = CONFIG.canvas.height / 3;  // 屏幕33%处
const followSpeed = 0.05;

// 修改后（远观）
const targetY = CONFIG.canvas.height * 0.4;  // 屏幕40%处
const followSpeed = 0.04;
```

**效果**：
- 熊猫位置更靠下（33% → 40%）
- 能看到更多上方和前方的场景
- 更慢的摄像机跟随速度，增强远观感

### 3. 熊猫初始位置调整
```javascript
// 修改前
this.panda = new Panda(200, 300);

// 修改后
this.panda = new Panda(200, 400);
```

**说明**：适应更大的画布高度

### 4. 雪花数量增加
```javascript
// 修改前
for (let i = 0; i < 150; i++) {

// 修改后
for (let i = 0; i < 200; i++) {
```

**说明**：更大的画布需要更多雪花来保持视觉密度
