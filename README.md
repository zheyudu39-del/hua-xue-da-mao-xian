# 🐼 熊猫滑雪大冒险 - Panda Ski Adventure

一个充满动漫风格的滑雪冒险游戏，玩家扮演一只可爱的熊猫在雪山上滑雪冒险！

## 🎮 游戏特色

### 核心玩法
- **动态坡度系统**：雪道坡度不断变化（缓坡、中坡、陡坡），模拟真实滑雪下坡体验
- **3D立体熊猫**：精美的3D熊猫角色，配备专业滑雪装备
  - 立体渐变效果
  - 护目镜和围巾装饰
  - 精美的红色滑雪板
  - 流畅的动画和阴影效果
- **简单控制系统**：
  - 按空格键：跳跃
  - 按住空格键：空中翻转加速（翻转时间越长，加速效果越强）
- **层次分明的背景**：多层山脉营造深度感和下坡视角
- **真实滑雪痕迹**：熊猫滑行时留下逐渐消失的雪道痕迹

### 障碍物系统（3D立体设计）
- **3D石头**：
  - 多面体立体效果
  - 真实的光影和高光
  - 裂纹细节
  - 可通过跳跃避开
- **3D小屋**：
  - 完整的3D建筑结构（墙体、屋顶、侧面）
  - 木板纹理和瓦片细节
  - 发光的窗户
  - 屋顶积雪效果
  - 无法跳过，可从屋顶或内部滑过
- 撞击10次后触发3D雪崩效果

### 动物伙伴（3D立体角色）
- **3D企鹅** 🐧：
  - 立体渐变身体
  - 可爱的翅膀和脚掌
  - 橙色渐变嘴巴
  - 弹跳动画效果
  - 骑上后获得加速
- **3D北极熊** 🐻：
  - 毛茸茸的立体效果
  - 细腻的五官和耳朵
  - 四肢动作
  - 骑上后获得更强加速
- 加速效果持续3秒，带金色粒子特效

### 视觉效果
- **3D立体渲染**：所有角色和物体都采用3D渐变和阴影效果
- **动态坡度变化**：雪道坡度实时变化，模拟真实滑雪场景
- **流畅粒子系统**：
  - 跳跃时的雪花粒子
  - 碰撞时的碎片效果
  - 加速时的金色粒子
  - 滑行时的雪道痕迹
- **多层次背景**：
  - 远景、中景、近景三层山脉
  - 动态飘落的雪花
  - 渐变天空效果
- **精美细节**：
  - 物体阴影和高光
  - 纹理和材质效果
  - 平滑的动画过渡

### 雪崩系统
- 碰撞10次后触发
- 雪崩会从后方追赶玩家
- 被雪崩追上游戏结束

## 🎯 游戏目标

- 尽可能滑得更远
- 避免碰撞障碍物
- 利用动物伙伴加速
- 掌握翻转技巧获得额外速度
- 在雪崩追上之前创造最高记录

## 🚀 如何开始

1. 打开 `index.html` 文件
2. 点击“开始游戏”按钮
3. 使用空格键控制熊猫跳跃

## 🎨 技术特点

- 纯原生 JavaScript 实现
- Canvas 2D 渲染
- 流畅的 60 FPS 动画
- 响应式设计
- 粒子系统
- 动态地形生成算法

## 📊 游戏统计

游戏会实时显示：
- **距离**：滑行的总距离
- **速度**：当前速度
- **碰撞次数**：已碰撞次数/最大碰撞次数

## 🎪 特效系统

- 跳跃时的雪花粒子
- 碰撞时的碎片效果
- 骑上动物时的金色粒子
- 翻转加速的视觉提示
- 雪崩的动态效果

## 🏆 游戏结束

游戏结束时会显示：
- 最终滑行距离
- 最高速度
- 总碰撞次数

## 💡 游戏技巧

1. **掌握跳跃时机**：提前跳跃避开石头
2. **利用翻转加速**：在大坡度时按住空格键进行翻转
3. **追逐动物**：主动接近企鹅和北极熊获得加速
4. **控制碰撞**：尽量避免碰撞，保持速度优势
5. **小屋策略**：可以选择从屋顶滑过或直接穿过
